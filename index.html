Testing Commands Reference:


# Authentication
curl -u student1:password123 http://yoursite/protected.html

# Cache Headers
curl -I http://yoursite/no-cache.html

# Compression

# Content Negotiation
curl -H "Accept: application/json" http://yoursite/resource

# Range Requests
curl -I -H "Range: bytes=0-99" http://yoursite/sample.pdf
